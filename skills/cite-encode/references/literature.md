# Cite ENCODE — Literature References

**Last updated:** 2026-03-07
**Purpose:** Reference catalog for the cite-encode skill — papers on ENCODE project publications, portal citation standards, and prepublication data sharing principles that guide proper attribution of ENCODE data.

---

### ENCODE Project Consortium 2012 — ENCODE Phase 2 integrative analysis

- **Citation:** ENCODE Project Consortium. An integrated encyclopedia of DNA elements in the human genome. *Nature*, 489(7414), 57-74, 2012.
- **DOI:** [10.1038/nature11247](https://doi.org/10.1038/nature11247)
- **PMID:** 22955616 | **PMC:** PMC3439153
- **Citations:** ~5,000
- **Key findings:** The flagship Phase 2 publication assigned biochemical functions to 80% of the human genome through systematic mapping of transcription, transcription factor binding, chromatin structure, and histone modification across 147 cell types. This is the most-cited ENCODE paper and remains the primary citation for any study that uses ENCODE data for regulatory element analysis. Any publication utilizing ENCODE annotations should cite this paper as the foundational resource.

Expanded encyclopaedias of DNA elements in the human and mouse genomes. *Nature*, 583(7818), 699-710, 2020.
- **DOI:** [10.1038/s41586-020-2493-4](https://doi.org/10.1038/s41586-020-2493-4)
- **PMID:** 32728249 | **PMC:** PMC7410828
- **Citations:** ~2,000
- **Key findings:** Reported 5,992 new datasets from Phase 3, including systematic determinations across mouse fetal development, and established a registry of 926,535 human and 339,815 mouse candidate cis-regulatory elements (cCREs). This is the current primary citation for ENCODE Phase 3 data and the cCRE catalog.

### Davis et al. 2018 — ENCODE portal citation standards

- **Citation:** Davis CA, Hitz BC, Sloan CA, Chan ET, Davidson JM, Gabdank I, Hilton JA, Jain K, Baymuradov UK, Narayanan AK, Onate KC, Graham K, Miyasato SR, Dreszer TR, Strattan JS, Jolanki O, Tanaka FY, Cherry JM. The Encyclopedia of DNA elements (ENCODE): data portal update. *Nucleic Acids Research*, 46(D1), D794-D801, 2018.
- **DOI:** [10.1093/nar/gkx1081](https://doi.org/10.1093/nar/gkx1081)
- **PMID:** 29126249 | **PMC:** PMC5753278
- **Citations:** ~400
- **Key findings:** Described the ENCODE Portal database and website design principles: public accessibility of protocols, analytical procedures, and data through a coherent search interface, alongside curated metadata recording data provenance. The portal houses over 13,000 datasets with standardized metadata and supports data from related projects including Roadmap Epigenomics and modENCODE. This paper documents the citation conventions and metadata standards that the cite-encode skill uses to generate proper attributions.

### Luo et al. 2020 — ENCODE portal updates with DOI system

- **Citation:** Luo Y, Hitz BC, Gabdank I, Hilton JA, Kagda MS, Lam B, Myers Z, Sud P, Jou J, Lin K, Baymuradov UK, Graham K, Litton C, Miyasato SR, Strattan JS, Jolanki O, Lee JW, Tanaka FY, Batchelder JE, Takeo I, Cherry JM. New developments on the Encyclopedia of DNA Elements (ENCODE) data portal. *Nucleic Acids Research*, 48(D1), D882-D889, 2020.
- **DOI:** [10.1093/nar/gkz1062](https://doi.org/10.1093/nar/gkz1062)
- **PMID:** 31713622 | **PMC:** PMC7145564
- **Citations:** ~800
- **Key findings:** Introduced the DOI system for ENCODE experiments and annotations, enabling stable, citable references to specific datasets. Also reported new data visualization tools, improved quality metrics displays, and expanded metadata for experiment provenance. The DOI system is fundamental to the cite-encode skill, as it provides permanent identifiers that can be included in publications to ensure reproducible data access.

### Toronto International Data Release Workshop 2009 — Prepublication data sharing principles

- **Citation:** Toronto International Data Release Workshop Authors. Prepublication data sharing. *Nature*, 461(7261), 168-170, 2009.
- **DOI:** [10.1038/461168a](https://doi.org/10.1038/461168a)
- **PMID:** 19741685
- **Citations:** ~1,500
- **Key findings:** Established the Toronto principles for prepublication data sharing in large-scale biological research projects, including the expectation that community resource datasets should be released rapidly and without restriction. These principles directly shaped ENCODE's data release policies, which make all data publicly available immediately upon quality review. Understanding these principles is essential for proper citation etiquette when using ENCODE data before associated publications appear.
