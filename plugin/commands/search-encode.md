description: Search ENCODE experiments by assay, organ, biosample, or target
---

Search the ENCODE Project for experiments matching the user's criteria.

Use the `encode_search_experiments` tool with filters like assay_title, organ, biosample_term_name, and target. Start with `encode_get_facets` if the user is exploring what data exists. Use `encode_get_metadata` to discover valid filter values.

Refer to the search-encode skill for detailed guidance.
